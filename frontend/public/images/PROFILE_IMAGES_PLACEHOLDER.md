# Profile Illustration Images

Please add the following profile illustration images to this directory:

## Required Images:

1. **profile-illustration.png** (Male/Other Dashboard)
   - A professional profile/user settings illustration
   - Should show a person managing their profile or settings
   - Recommended size: 800x400px or similar aspect ratio
   - Should match the theme and style of chat-illustration.png

2. **profile-illustration-female.png** (Female Dashboard)
   - A professional profile/user settings illustration
   - Should show a female person managing their profile or settings
   - Recommended size: 800x400px or similar aspect ratio
   - Should match the theme and style of chat-illustration-female.png

## Notes:
- Images should be realistic, not cartoon-like or toy-like
- Should have a professional, modern look
- Will be displayed with object-cover and rounded corners (2rem top)
- Height will be constrained to 16rem (h-64)

## Usage:
These images will be automatically displayed on the Profile Settings cards in the dashboard based on the selected gender.
